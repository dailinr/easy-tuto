# easy-tuto

La idea de una plataforma web que sirva para agilizar el proceso de comunicación entre los estudiantes y monitores académicos de la universidad del magdalena.

Al lograrse con su correo institucional, al estudiante le aparecerá un listado de las materias que actualmente está cursando en el semestre, así también los monitores disponibles para esta materias (si los hay). 

Será mucho más fácil poder llegar a un acuerdo dependiendo de los horarios de las personas que intervienen.
